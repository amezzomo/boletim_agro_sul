
Boletim Agro Sul - Projeto de Site (Template)
============================================

Conteúdo pronto para upload em um repositório GitHub / Netlify.

Estrutura:
- index.html           -> página principal (conteúdo editorial + gráficos)
- assets/logo.svg      -> logo oficial (SVG)
- css/styles.css       -> estilos base com variáveis de cor
- js/charts.js         -> inicialização de gráficos (usa Chart.js CDN)
- sample_data.json     -> exemplo de dados usado no boletim

Como usar:
1) Faça upload desta pasta para um repositório GitHub ou Netlify.
2) Abra index.html ou configure o deploy no Netlify apontando para a pasta.
3) Substitua os textos e dados em index.html / sample_data.json conforme cada edição semanal.
4) Para modificar cores, ajuste as variáveis em css/styles.css.
